Acest repository conține un proiect universitar (Baze de Date) care simulează sistemul de gestiune al unei firme de construcții specializate în realizarea terenurilor de fotbal. Aplicația este de tip full-stack, având baza de date modelată în Oracle SQL, un API creat cu Node.js (Express) și o interfață web creată cu HTML/CSS/JS pentru vizualizarea, filtrarea și modificarea datelor (CRUD).
